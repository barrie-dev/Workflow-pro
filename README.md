# WorkFlow Pro

Kant-en-klare SaaS demo voor Belgische KMO's met veldpersoneel.

## Starten

Snelste optie: dubbelklik op:

```text
Open WorkFlow Pro.bat
```

Server-optie: dubbelklik op:

```text
Start WorkFlow Pro.bat
```

Of start via terminal met:

```powershell
npm start
```

Open daarna:

```text
http://localhost:4173
```

## Wat erin zit

- Multi-tenant platformdashboard voor super admin
- Tenant admin, werfleider en medewerker rollen
- Planning en taken per locatie
- Prikklok en tijdregistraties
- Onkosten met goedkeuringen
- Digitale werkbonnen met checklist, materiaal en bestanden
- Verlof, berichten, stock en wagenpark
- Rapportages, integraties en SaaS billing flow
- Server-side opslag wanneer gestart via `Start WorkFlow Pro.bat`
- Lokale browseropslag als fallback bij direct openen

## Demo-login
